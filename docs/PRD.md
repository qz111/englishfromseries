Immersive English Learning Video Player
Platform: Windows Desktop Application
Target Audience: Intermediate to Advanced English learners.
Core Value: A local media player that seamlessly integrates Whisper ASR and on-demand LLM analysis to create an English-only, highly contextual learning loop (Watch -> Explicit Mark -> Pinpoint Diagnose -> Review).

1. Module: Media & Local Pre-processing
Local Import: Users drag and drop local video files (.mp4, .mkv).

Audio Extraction: The app uses an embedded lightweight FFmpeg to silently extract the audio track locally (e.g., into .mp3 or .wav).

ASR Processing (Whisper): The extracted audio is sent to a Whisper model (local or API) to generate the baseline transcript.

Crucial Requirement: Whisper MUST return both sentence-level and word-level timestamps. The word-level data is kept dormant until the user needs to select a specific word in Step 3.

2. Module: Playback & Explicit Capture (First Pass)
Immersive Viewing: Users watch the video without subtitles.

Explicit Capture (Sentence-Level): When a user encounters a "blind spot" (doesn't understand what was just said), they press a dedicated button (specially designed for this player).

Action: The app instantly flags the entire current sentence as isMarkedByUser: true.

No interruption: The video continues playing smoothly. User pause the video and click the button is all right.

No LLM calls: Absolutely no AI text generation happens at this stage.

3. Module: Smart Diagnostic Transcript (Second Pass)
Review Mode: The user opens the side panel containing the scrolling English transcript. Sentences marked during the First Pass are visually highlighted.

Targeted Diagnosis: When the user clicks a marked sentence, playback pauses, and a diagnostic menu appears.

Option A: "Pronunciation / Linking" (No LLM Call)

If the user selects this, the app simply tags this sentence for audio review (pronunciationFlagged: true). It will be added to the final review deck so the user can repeatedly loop the exact audio snippet.

Option B: "Vocabulary / Slang" (Contextual LLM Call)

The user is prompted to highlight/click the specific word or phrase inside the sentence that confused them.

API Trigger: The app sends the highlighted word AND the entire sentence (for context maybe also last sentence and next sentence) to the LLM.

LLM Output: A pure English, concise explanation of that specific word based strictly on the provided context, plus an example sentence.

4. Module: Review & Export Center
Audio Loop Deck: A dedicated section where users can play back all sentences flagged for "Pronunciation" in an infinite loop.

Contextual Notes: An exportable document (Markdown/PDF) containing the original sentences, the exact words the user queried, and the AI's English-only explanations.

5. Reference Data Structure (Progressive Hydration)
This JSON structure is designed to be updated progressively. Claude Code should use this as the primary state management model for the transcript.

JSON
{
  "transcript": [
    {
      "sentenceId": "s_001",
      "startTime": 12.5,
      "endTime": 15.2,
      "text": "I mean, it's a moo point. It's like a cow's opinion.",
      
      // STEP 2: Turns to true ONLY when the user presses the hotkey during playback.
      "isMarkedByUser": true, 
      
      // STEP 1: Generated by Whisper. Needed for UI highlighting in Step 3.
      "words": [
        {"word": "I", "start": 12.5, "end": 12.6},
        {"word": "mean", "start": 12.6, "end": 12.8},
        {"word": "it's", "start": 12.8, "end": 13.0},
        {"word": "a", "start": 13.0, "end": 13.1},
        {"word": "moo", "start": 13.1, "end": 13.4},
        {"word": "point", "start": 13.4, "end": 13.8}
        // ...
      ],
      
      // STEP 3: Updated based on user interaction in the Diagnostic Menu.
      "diagnostics": {
        // If user clicks "Pronunciation", this turns true. NO LLM text needed.
        "pronunciationFlagged": true, 
        
        // If user highlights "moo point" and clicks "Vocabulary/Slang", app calls LLM and pushes the result here.
        "vocabularyQueries": [
          {
            "selectedWord": "moo point",
            "aiExplanation": "In this context, a 'moo point' is a humorous mispronunciation of 'moot point'. It means a point that is completely irrelevant."
          }
        ]
      }
    }
  ]
}
